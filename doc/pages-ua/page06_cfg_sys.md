# Конфігурування системи modbusua {#sec_cfg_sys}

Конфігурація сервера modbusua задається у секції `[SYSTEM]` файлу конфігурації `modbusua.conf`.

Список налаштувань конфігурації системи:

| Назва                     | Опис                                                              
|:--------------------------|:------------------------------------------------------------------
| `Name`                    | Ім’я проекту системи                                              
| `include`                 | Ім'я іншого файлу конфігурації, що включається і розширяє даний   
| `Log.flags`               | Список прапорів, що задають рівень логування                      
| `Log.output`              | Список типів журналювання для виводу (див. нижче)                 
| `Log.default.format`      | Формат одного повідомлення за замовчуванням (див. нижче)          
| `Log.default.timeformat`  | Формат часу одного повідомлення за замовчуванням (див. нижче)     
| `Log.console.format`      | Формат одного повідомлення для виводу у консоль (див. нижче)      
| `Log.console.timeformat`  | Формат часу одного повідомлення для виводу у консоль (див. нижче) 
| `Log.file.format`         | Формат одного повідомлення для виводу у файл (див. нижче)         
| `Log.file.timeformat`     | Формат часу одного повідомлення для виводу у файл (див. нижче)    
| `Log.file.path`           | Розташування файлу, у який будуть виводитись повідомлення         
| `Log.file.maxcount`       | Максимальна кількість файлів для ротації журналу                  
| `Log.file.maxsize`        | Максимальний розмір файла ротації журналу (в байтах). Підтримуються суфікси: `KB`, `MB`, `GB` (наприклад, `5 MB`)              

Опис прапорів `Log.flags` наведений у розділі [Журналювання](#sec_journal_common).

## Конфігурування протоколу OPC UA {#sec_cfg_sys_ua}

Конфігурація протоколу OPC UA сервера modbusua задається у секції `[OPCUA]` файлу конфігурації `modbusua.conf`.

| Назва          | Опис                                                                          |
|:---------------|:------------------------------------------------------------------------------|
| `Port`         | Номер TCP порту на якому працює сервер (за замовч. '4840')                    |
| `UncertainAs`  | Розглядати значення `Uncertain` як: `Good`, `Bad`, `Uncertain` (за замовч.)   |
