# Загальний опис {#sec_descr_common}

Сервер modbusua призначений для збору інформації із існуючих пристроїв через протокол Modbus та
подальшої їх обробки і передачі через загальновідомий і загальновживаний протокол OPC UA
(Open Platform Communications Unified Architecture).
