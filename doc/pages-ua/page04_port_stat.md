# Статистика/діагностика порту {#sec_port_stat}

Використовується для діагностики роботи gateway. Дані `ItemReference` є ReadOnly.

| `ItemReference`          | Тип даних     | Призначення                                                                                       |
|:-------------------------|:--------------|:--------------------------------------------------------------------------------------------------|
| `ThreadCycleCount`       | `UInt32`      | Виконання потоку. Кількість циклів від початку ведення статистики                                 |
| `ThreadLastCycle`        | `UInt32`      | Виконання потоку. Час виконання останнього циклу (мікросекунди)                                   |
| `ThreadMinCycle`         | `UInt32`      | Виконання потоку. Мінімальний час виконання циклу від початку ведення статистики (мікросекунди)   |
| `ThreadMaxCycle`         | `UInt32`      | Виконання потоку. Максимальний час виконання циклу від початку ведення статистики (мікросекунди)  |
| `ThreadAvgCycle`         | `UInt32`      | Виконання потоку. Середній час виконання циклу від початку ведення статистики (мікросекунди)      |
| `GoodCount`              | `UInt32`      | Загальна кількість успішних Modbus-посилок                                                        |
| `BadCount`               | `UInt32`      | Загальна кількість неуспішних Modbus-посилок                                                      |
| `Status`                 | `ModbusStatus`| Статус останньої Modbus-посилки (\ref sec_data_statusModbus)                                      |
| `Timestamp`              | `Timestamp`   | Час (PC) останньої Modbus-посилки                                                                 |
| `LastSuccessTimestamp`   | `Timestamp`   | Час (PC) останньої успішної Modbus-посилки                                                        |
| `LastErrorStatus`        | `ModbusStatus`| Статус останньої помилки Modbus-посилки (\ref sec_data_statusModbus)                              |
| `LastErrorTimestamp`     | `Timestamp`   | Час (PC) останньої неуспішної Modbus-посилки                                                      |
| `LastErrorText`          | `String`      | Текстове представлення останньої помилки Modbus-посилки                                           |
