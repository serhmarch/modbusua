# Команди пристрою/ПЛК {#sec_device_cmd}

gateway може приймати команди від верхнього рівня під час роботи.
Команди представляють собою `ItemReference` типу `Bool`, 
в який при записі в нього значення  `=true` команда виконується або додається в буфер для подальшого виконання. 
Даний тип `ItemReference` призначений лише для запису і в разі читання завжди повертає `false`.

Список команд для пристрою/ПЛК:

| `ItemReference`       | Призначення                                   |
|:----------------------|:----------------------------------------------|
| `StatClear`           | Онулення статистичної інформації              |
