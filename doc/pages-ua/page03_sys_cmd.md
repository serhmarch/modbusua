# Системні команди modbusua {#sec_sys_cmd}

gateway може приймати команди від верхнього рівня під час роботи. Команди представляють собою item reference типу `Bool`, 
в який при записі в нього значення  `=true` команда виконується або додається в буфер для подальшого виконання. 
Даний тип item reference призначений лише для запису і в разі читання завжди повертає `false`.

Список команд для modbusua:

| `ItemReference` команди   | Призначення                            |
|:--------------------------|:---------------------------------------|
| `ReloadConfig`            | Перезавантаження конфігурації із файла |
