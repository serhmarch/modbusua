# Налаштування порту {#sec_port_set}

gateway має змогу змінювати певні параметри під час  виконання програми. 
Дані `ItemReference` призначені як для зчитування так і для запису (Read/Write).

| `ItemReference`   |	Тип даних   | Призначення                                                                   |
|:------------------|:--------------|:------------------------------------------------------------------------------|
| `Enable`          | `Bool`        | Доступність даного порту для циклу виконання (1-доступний, 0-недоступний). Якщо порт стає недоступним, він автоматично закривається (розривається з’єднання)
