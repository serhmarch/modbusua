# Data Type Suffixes {#sec_data_suffix}

Data type suffixes define the `ItemReference` size. Used for status registers and Modbus registers.

| Suffix          | Data Type          |
|:----------------|:-------------------|
| `S`             | `Int16`            |
| `R`             | `UInt16`           |
| `I`             | `Int32`            |
| `L`             | `Int32`            |
| `U`             | `UInt32`           |
| `LL`            | `Int64`            |
| `UL`            | `UInt64`           |
| `F`             | `Float`            |
| `LF`            | `Double`           |
| `IS`            | `SwappedInt32`     |
| `LS`            | `SwappedInt32`     |
| `US`            | `SwappedUInt32`    |
| `LLS`           | `SwappedInt64`     |
| `ULS`           | `SwappedUInt64`    |
| `FS`            | `SwappedFloat`     |
| `LFS`           | `SwappedDouble`    |
