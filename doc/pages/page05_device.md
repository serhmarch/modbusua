# Об'єкт пристрій/ПЛК {#sec_device_common}

Об’єкт пристрій включає в себе налаштування конкретного пристрою/ПЛК. 
Даний об'єкт-пристрій зв'язаний з конкретним віддаленим ПЛК.
При цьому пристрій конфігурується для роботи в Modbus мережі: задається стандартна Modbus-адреса пристрою,
максимальна кількість бітів/регістрів для читання однією посилкою тощо.

Таких об’єктів пристроїв для одного порту може бути безліч, і відповідно
обмін із кожним пристроєм буде виконуватись послідовно. 
Таким чином використувуючи один порт та декілька пристроїв можна будувати стандартну Modbus-мережу.

В наступних розділах описаний доступ до елементів порту (структур даних) через механізм `ItemReference`, 
які дозволяють отримувати дані про роботу даного пристрою а також змінювати його поведінку.
