# Налаштування пристрою/ПЛК {#sec_device_set}

Драйвер має змогу змінювати певні параметри під час  виконання програми. 
Дані `ItemReference` призначені як для зчитування так і для запису (Read/Write).

| `ItemReference`   | Тип даних | Призначення                                                                       |
|:------------------|:----------|:----------------------------------------------------------------------------------|
| `EnableDevice`    | `Bool`    | Доступність даного пристрою/ПЛК для циклу виконання (1-доступний, 0-недоступний)  |
