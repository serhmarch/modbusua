# Команди порту {#sec_port_cmd}

Драйвер може приймати команди від верхнього рівня під час роботи. Команди представляють собою item reference типу `Bool`, 
в який при записі в нього значення  `=true` команда виконується або додається в буфер для подальшого виконання. 
Даний тип item reference призначений лише для запису і в разі читання завжди повертає `false`.

Список команд для порту:

| `ItemReference` команди   | Призначення                       |
|:--------------------------|:----------------------------------|
| `StatClear`               | Онулення статистичної інформації  |
